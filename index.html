Commandes vocales

Les commandes vocales vous permettent de contrôler l'application via votre voix. Après avoir cliqué sur Activer, le guide commencera à écouter vos commandes vocales et poursuivra après chaque réinitialisations jusqu'à ce que vous la désactiviez. Les commandes seront écoutées à partir de votre microphone par défaut, alors assurez-vous qu'il soit correctement configuré. La reconnaissance vocale continuera à fonctionner même si la page n'est pas nette et que vous utilisez le même microphone dans le jeu.
Preuves :
  • "nombre de preuves <num>" ou "difficulté <num>" -> Régler le nombre de preuves
  • "preuve <preuve>" -> Marquer comme preuve
  • "preuve pas <preuve>" -> Marquer comme n'étant pas une preuve
  • "preuve annuler <preuve>" -> Réinitialiser l'état de la preuve
  • "patte de singe <preuve>" -> Marquer la preuve avec la logique de la patte de singe
Vitesse :
  • "vitesse <vitesse>" -> Marquer la vitesse spécifiée
  • "vitesse effacer <vitesse>" -> Réinitialiser l'état de la vitesse spécifiée
  • "vitesse ligne de vue" -> Marque l'entité comme accélérant dans ligne de vue
  • "vitesse pas ligne de vue" -> Marque l'entité comme n'accélérant pas dans ligne de vue
  • "vitesse annuler ligne de vue" -> Réinitialise le filtre "ligne de vue"
Seuil de chasse :
  • "seuil de chasse <santé mentale>" -> Inscrire le seuil de chasse spécifié
  • "seuil de chasse effacer <santé mentale>" -> Réinitialiser l'état du seuil de chasse spécifié
Entités :
  • "entité pas <entité>" -> Griser le type d'entité spécifié
  • "entité annuler <entité>" -> Réinitialiser l'état de type d'entité (hors types cachés)
  • "entité choisir <entité>" -> Sélectionner/mettre en surbrillance le type d'entité spécifié
  • "entité rayer <entité>" -> Supprimez le type d'entité spécifié de l'écran
  • "entité voir|info <entité>" -> Faire défilier la carte de l'entité spécifiée dans la vue
Minuteur :
  • "minuteur démarrer" -> Démarrer le minuteur de trace
  • "minuteur arrêter " -> Arrêter le minuteur de trace
  • "cooldown démarrer" -> Démarrer le minuteur de cooldown/temps de recharge
  • "cooldown arrêter " -> Arrêter le minuteur de cooldown/temps de recharge
Divers :
  • "afficher filtres" ou "afficher outils" -> Modifier le menu pour afficher les filtres/outils
  • "vitesse entité <pourcentage>" -> Régler le modificateur de vitesse des fantômes sur 50 %, 75 %, 100 %, 125 % ou 150 %
  • "voir carte [<nom de la carte>]" -> Ouvrir l'onglet carte et afficher cette carte
  • "fermer carte" ou "cacher carte" -> Fermer l'onglet carte
  • "réinitialiser journal" -> Réinitialiser l'aide-mémoire (sauvegardé s'il est lié à Discord)
  • "arrêter reconnaissance vocale" -> Arrêtez la reconnaissance vocale

Confidentialité des données

Les commandes vocales sont acheminées via l'API Web Speech intégrée. Bien que l'accès au microphone soit requis pour cette fonctionnalité, votre voix n'est jamais enregistrée ou stockée.
